# Mwasia-score
# ⚡ Mwa$ia – Questionnaire Terrain

Ce projet contient le **questionnaire de terrain Mwa$ia**, conçu pour comprendre les réalités économiques des femmes et identifier leur capacité à lancer une activité génératrice de revenus.

---

## 🎯 Objectif

- Collecter des données terrain simples
- Identifier les blocages (argent, temps, idée…)
- Évaluer le potentiel entrepreneurial
- Orienter automatiquement vers Mwa$ia

---

## 🚀 Fonctionnement

1. L’utilisatrice remplit le questionnaire
2. Les réponses sont envoyées vers Google Sheets
3. Un score est calculé automatiquement
4. Un profil est attribué :
   - 🔥 PRETE
   - ⚡ POTENTIEL
   - 🧠 A_STRUCTURER
5. Si le profil est **PRETE**, elle est redirigée vers WhatsApp
